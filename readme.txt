== Description ==

Displays a Micheal Jackson memorial badge in the sidebar of your blog via widget interface or anywhere else via function call. Make the people remember the King of Pop.


== Installation ==

1. Unpack the zipfile kingofpop-X.y.zip
1. Upload folder `kingofpop` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. configure the color etc. via 'admin interface'
1. Place `<?php if(function_exists('kingofpop_display')) kingofpop_display(); ?>` in your template or use the sidebar widgets.
